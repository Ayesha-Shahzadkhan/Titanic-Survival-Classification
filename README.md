# Titanic Survival Classification

A Machine Learning project that predicts whether a Titanic passenger 
survived or not based on features like age, gender, ticket class, and fare.

## Features
- Logistic Regression model with 81% accuracy
- Label Encoding for categorical data processing
- Interactive live prediction using Streamlit

## Dataset
891 real passenger records from the Titanic (Kaggle Dataset).
